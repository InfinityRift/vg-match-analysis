    • Where did this come from?

    I made this site because I thought there were a lot of cool data that can be gleaned from the telemetry information that Super Evil Megacorp and MadGlory recently released to the developer community. It's not as polished as I'd like, because I threw it together in one week.

    I'm a Vainglory player of no decent skill tier at all, and I hope some of these robots I built can coach me! I sometimes do Vainglory analytics videos over here, although not as recently as I would like.

    But I really enjoyed seeing the kinds of analyses we can do with even the basic data we have now. And I'm also very excited to see what other developers in the Vainglory API Discord have been doing, because there are tons of cool ideas there as well.

    Speaking of which, one of them is Dan, who built a very nice JavaScript Vainglory API Client that I used in this project, and which several others are using in their work as well.
